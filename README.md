# Task Management Application

A simple Task Management Web Application built using **HTML, CSS, JavaScript, Python (Flask), and SQLite**.

## Features

- User Registration
- User Login
- User Logout
- Create Tasks
- View Tasks
- Update Task Status
- Delete Tasks
- Responsive Design
- SQLite Database

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Python
- Flask
- SQLite

## Installation

Install dependencies

```bash
pip install -r requirements.txt
```

Run the application

```bash
python app.py
```

Open your browser

```
http://127.0.0.1:5000
```
